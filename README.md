# Aula: Introdução à Blockchain e Ethereum Smart Contracts

Introdução à Blockchain e Ethereum Smart Contracts

## Instalação

```bash
$ npm install
```

## Rodando o projeto

```bash
$ npm start
```

## Tecnologies used in the project
1. NodeJS
2. Web3
3. Remix
4. Ganache
